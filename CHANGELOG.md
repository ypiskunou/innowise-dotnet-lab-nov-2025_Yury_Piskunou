# Changelog

Все изменения в проекте InnoShop.

## [0.2.0] - 2025-12-08
### Добавлено (Frontend)
- **Личный кабинет:** Страница `/my-products` для управления своими товарами.
- **Редактирование:** Возможность изменять товары и менять статус (Активен/Черновик).
- **Живой поиск:** Компонент Typeahead в шапке сайта для быстрого перехода к товару.
- **Сайдбар категорий:** Выезжающая панель с фильтрацией и счетчиками товаров (Glassmorphism UI).
- **Авто-разлогин:** Перехватчик Axios для автоматического выхода при истечении токена (401).

### Добавлено (Backend)
- **ProductService:**
  - Эндпоинт `GET /api/products/my` (игнорирует глобальные фильтры для владельца).
  - Эндпоинт `GET /api/products/previews` (оптимизированная проекция для поиска).
  - Поля `IsActive`, `CategoryId` в `ProductDto`.
  - Защита от просмотра скрытых товаров по прямой ссылке (проверка владельца).
- **UserService:**
  - Защита от деактивации самого себя для Администратора.
  - Проброс токена авторизации при межсервисных запросах (HttpClient).

### Исправлено
- Исправлен баг, когда Админ не видел забаненных пользователей в списке (добавлен фильтр).
- Исправлена ошибка CORS при работе с локальными портами.
- Исправлен маппинг ролей в `UserDto` (теперь возвращаются строки, а не объекты).
